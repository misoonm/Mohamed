دليل إعدادات الأمان في Windows

دليل شامل لإعداد سياسات المجموعة وإدارة المستخدمين والمجموعات لتعزيز أمان نظامك

مقدمة عن أدوات إدارة الأمان في Windows

يقدم نظام Windows أدوات قوية لإدارة إعدادات الأمان والسياسات التي تتحكم في سلوك النظام وتمنحك التحكم في أمان جهازك. من خلال هذه الأدوات، يمكنك تعزيز أمان نظامك عن طريق إدارة كلمات المرور، وضبط سياسات قفل الحساب، وإدارة المستخدمين والمجموعات، وتعيين حقوق المستخدم، وضبط خيارات الأمان المتقدمة.

سيرشدك هذا الدليل خطوة بخطوة لاستخدام أداتين رئيسيتين: محرر نهج المجموعة المحلي (gpedit.msc) والمستخدمون والمجموعات المحليون (lusrmgr.msc) لتحسين أمان نظامك.

الوصول إلى الأدوات اللازمة

فتح "Local Group Policy Editor"

اضغط على مفتاحي Windows + R لفتح نافذة "تشغيل".

اكتب gpedit.msc واضغط على Enter.

صورة توضيحية: نافذة Run مع كتابة gpedit.msc

فتح "Local Users and Groups"

اضغط على مفتاحي Windows + R لفتح نافذة "تشغيل".

اكتب lusrmgr.msc واضغط على Enter.

صورة توضيحية: نافذة Run مع كتابة lusrmgr.msc

سياسات كلمات المرور (Password Policy)

فرض تأريخ كلمة المرور

الموقع: gpedit.msc > Computer Configuration > Windows Settings > Security Settings > Account Policies > Password Policy

انقر نقرًا مزدوجًا على "Enforce password history".

اختر "Enabled" وحدد عدد كلمات المرور التي يجب تذكرها (مثلاً 12).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Enforce password history

الحد الأقصى لعمر كلمة المرور

انقر نقرًا مزدوجًا على "Maximum password age".

اختر "Enabled" وحدد عدد الأيام لتغيير كلمة المرور (مثلاً 90 يومًا).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Maximum password age

الحد الأدنى لطول كلمة المرور

انقر نقرًا مزدوجًا على "Minimum password length".

اختر "Enabled" وحدد الحد الأدنى للأحرف (مثلاً 8 أحرف).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Minimum password length

متطلبات تعقيد كلمة المرور

انقر نقرًا مزدوجًا على "Password must meet complexity requirements".

اختر "Enabled" لفرض استخدام أحرف كبيرة وصغيرة وأرقام ورموز.

اضغط Apply ثم OK.

صورة توضيحية: إعداد Password complexity requirements

سياسات قفل الحساب (Account Lockout Policy)

حد قفل الحساب

الموقع: gpedit.msc > Computer Configuration > Windows Settings > Security Settings > Account Policies > Account Lockout Policy

انقر نقرًا مزدوجًا على "Account lockout threshold".

اختر "Enabled" وحدد عدد المحاولات الفاشلة (مثلاً 3).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Account lockout threshold

مدة قفل الحساب

انقر نقرًا مزدوجًا على "Account lockout duration".

حدد المدة التي يظل فيها الحساب مقفلاً (مثلاً 30 دقيقة).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Account lockout duration

إعادة تعيين عداد القفل

انقر نقرًا مزدوجًا على "Reset account lockout counter after".

حدد المدة قبل إعادة تعيين العداد (مثلاً 30 دقيقة).

اضغط Apply ثم OK.

صورة توضيحية: إعداد Reset account lockout counter

المستخدمون والمجموعات المحليون (Local Users and Groups)

إنشاء مستخدم جديد

الموقع: lusrmgr.msc

انقر بزر الفأرة الأيمن على "Users" واختر "New User".

أدخل اسم المستخدم وكلمة المرور وتأكيدها.

ألغِ تحديد "User must change password at next logon" وحدد "Password never expires" إذا لزم الأمر.

اضغط Create.

صورة توضيحية: نافذة إنشاء مستخدم جديد

إنشاء مجموعة جديدة

انقر بزر الفأرة الأيمن على "Groups" واختر "New Group".

أدخل اسم المجموعة.

اضغط Create.

صورة توضيحية: نافذة إنشاء مجموعة جديدة

إضافة مستخدم إلى مجموعة

انقر نقرًا مزدوجًا على المجموعة التي أنشأتها.

انقر على "Add".

اكتب اسم المستخدم (مثلاً user1) واضغط "Check Names" ثم OK.

صورة توضيحية: إضافة مستخدم إلى مجموعة

تعيين حقوق المستخدم (User Rights Assignment)

إيقاف تشغيل النظام

الموقع: gpedit.msc > Computer Configuration > Windows Settings > Security Settings > Local Policies > User Rights Assignment

انقر نقرًا مزدوجًا على "Shut down the system".

انقر على "Add User or Group" لإضافة المستخدمين أو المجموعات.

اضغط Apply ثم OK.

صورة توضيحية: إعداد Shut down the system

تغيير وقت النظام

انقر نقرًا مزدوجًا على "Change the system time".

يمكنك إزالة مجموعة "Users" لمنع المستخدمين العاديين من تغيير الوقت.

اضغط Apply ثم OK.

صورة توضيحية: إعداد Change the system time

الوصول إلى هذا الكمبيوتر من الشبكة

انقر نقرًا مزدوجًا على "Access this computer from the network".

أضف أو أزل المستخدمين والمجموعات للتحكم في من يمكنه الاتصال بالكمبيوتر عبر الشبكة.

اضغط Apply ثم OK.

صورة توضيحية: إعداد Access this computer from the network

خيارات الأمان (Security Options)

عدم عرض آخر اسم مستخدم

الموقع: gpedit.msc > Computer Configuration > Windows Settings > Security Settings > Local Policies > Security Options

انقر نقرًا مزدوجًا على "Interactive logon: Do not display last user name".

اختر "Enabled" لمنع عرض اسم المستخدم الأخير.

اضغط Apply ثم OK.

صورة توضيحية: إعداد عدم عرض آخر اسم مستخدم

نص رسالة تسجيل الدخول

انقر نقرًا مزدوجًا على "Interactive logon: Message text for users attempting to log on".

اكتب رسالة تحذيرية تظهر للمستخدمين قبل تسجيل الدخول.

اضغط Apply ثم OK.

صورة توضيحية: إعداد رسالة تسجيل الدخول

حالة حساب المسؤول

انقر نقرًا مزدوجًا على "Accounts: Administrator account status".

اختر "Disabled" لتعطيل حساب المسؤول الافتراضي.

اضغط Apply ثم OK.

صورة توضيحية: تعطيل حساب المسؤول

ملاحظة هامة

بعد تطبيق أي من هذه السياسات، يجب تحديث نهج المجموعة لتفعيل التغييرات على الفور. يمكنك القيام بذلك عبر تشغيل نافذة الأوامر كمسؤول وكتابة الأمر التالي:

gpupdate /force

هذا الأمر سيؤكد تطبيق جميع التغييرات التي أجريتها على سياسات المجموعة فوراً دون الحاجة لإعادة تشغيل النظام.
