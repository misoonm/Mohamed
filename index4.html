تصميم سكربت PowerShell آمن لاختبار بيئات أمنية

تحميل السكربت

محاكاة آمنة داخل بيئة معزولة

فكرة المشروع

هذا السكربت يحاكي سلوك برمجية خبيثة تستهدف ملفات Excel وAccess، لكنه لا يحذف أي ملفات حقيقية، بل ينسخها إلى مجلد احتياطي ويسجل كل عملية في ملف CSV. مخصص للاستخدام في بيئة معزولة (VM) لغرض التعليم والاختبار.

كود PowerShell 


# SafeSimulation_MoveToBackup.ps1
param(
    [string]$TargetFolder = "C:\Users\Public\Documents\TestFiles",
    [string]$BackupFolder = "C:\Users\Public\Documents\Backup_Sim",
    [string]$LogFile = "C:\Users\Public\Documents\sim_log.csv"
)

if (-not (Test-Path $BackupFolder)) {
    New-Item -ItemType Directory -Path $BackupFolder | Out-Null
}

$patterns = "*.xls","*.xlsx","*.accdb"
$results = @()

foreach ($p in $patterns) {
    $files = Get-ChildItem -Path $TargetFolder -Filter $p -File -Recurse -ErrorAction SilentlyContinue
    foreach ($f in $files) {
        $dest = Join-Path $BackupFolder ($f.BaseName + "_SIMCOPY" + $f.Extension)
        Copy-Item -Path $f.FullName -Destination $dest -Force
        $entry = [PSCustomObject]@{
            Timestamp    = (Get-Date).ToString("s")
            OriginalPath = $f.FullName
            BackupPath   = $dest
            SizeBytes    = $f.Length
        }
        $results += $entry
    }
}

if ($results.Count -gt 0) {
    $results | Export-Csv -Path $LogFile -NoTypeInformation -Append
    Write-Output "Simulation completed. $($results.Count) files copied. Log: $LogFile"
} else {
    Write-Output "No target files found under $TargetFolder"
}

خطوات التنفيذ

  1. إنشاء بيئة افتراضية (VM) معزولة عن الإنترنت.
  2. إنشاء مجلد TestFiles ووضع ملفات تجريبية فيه.
  3. حفظ السكربت باسم SafeSimulation_MoveToBackup.ps1.
  4. تشغيل PowerShell كمسؤول وتنفيذ: 
    Set-ExecutionPolicy Bypass -Scope Process -Force
.\SafeSimulation_MoveToBackup.ps1
  5. مراجعة مجلد النسخ الاحتياطي وملف sim_log.csv للتحليل.

النتائج والتحليل

بعد التنفيذ، ستحصل على نسخ آمنة من الملفات المستهدفة وسجل زمني لعمليات النسخ. يمكن تحليل هذا السجل باستخدام Excel أو أي أداة تحليل بيانات لمعرفة سلوك المحاكاة.
